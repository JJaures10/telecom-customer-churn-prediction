#  Telecom Customer Churn Prediction
**A Machine Learning approach to reducing customer attrition.**

## 🎯 Project Objective
The goal of this project is to build a predictive model that identifies which customers are likely to churn. By pinpointing high-risk accounts, the marketing team can offer targeted retention incentives, potentially saving the company thousands in lost revenue.

## 🛠️ Data & Methodology
- **Data Source:** [Kaggle Telco Churn Dataset](https://kaggle.com)
- **Problem Type:** Binary Classification (Churn vs. No Churn)
- **Key Techniques:** Feature Engineering, Handling Class Imbalance, Hyperparameter Tuning.

## 📁 Repository Structure
- `data/`: Raw and cleaned data files.
- `notebooks/`: Step-by-step EDA and model building.
- `reports/`: Final visualizations and performance metrics.
